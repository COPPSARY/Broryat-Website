Two-Step Verification បន្ថែម password មួយលើសពីលេខកូដ SMS ឬ OTP។ វាជួយការពារគណនី បើអ្នកណាម្នាក់អាចទទួលបានលេខកូដ login របស់អ្នក។

## ជំហានបើក តាម Telegram

### 1. បើក `Settings`

នៅក្នុង Telegram សូមចូលទៅកាន់ផ្ទាំង `Settings` ជាមុនសិន។

![Telegram settings step](/assets/articles/2Factor/step-1-settings.jpg)

### 2. ជ្រើស `Privacy and Security`

បន្ទាប់មក ចុច `Privacy and Security` ដើម្បីចូលទៅកាន់ផ្នែកសុវត្ថិភាពគណនី។

![Privacy and Security step](/assets/articles/2Factor/step-2-privacy&security.jpg)

### 3. បញ្ចូល password សម្រាប់ Two-Step Verification

Telegram នឹងស្នើឱ្យអ្នកកំណត់ password ថ្មីមួយ។ សូមជ្រើស password ដែលខុសពី PIN និង password ផ្សេងៗដែលអ្នកកំពុងប្រើ។

![Set password step](/assets/articles/2Factor/step-3-put-password.jpg)

## សេចក្តីសង្ខេប

1. បើក Telegram ហើយចូលទៅ `Settings`
2. ជ្រើស `Privacy and Security`
3. ចុច `Two-Step Verification`
4. បង្កើត password ដែលខុសពី PIN ឬ password ផ្សេងទៀត
5. បន្ថែម email សង្គ្រោះ ប្រសិនបើអាចធ្វើបាន

## បន្ទាប់ពីបើករួច

កុំចែករំលែក password នេះជាមួយអ្នកណា។ ប្រសិនបើមានសារអះអាងថាត្រូវការពាក្យសម្ងាត់នេះ សូមចាត់ទុកវាជាសញ្ញាបោកប្រាស់ជាមុនសិន។
