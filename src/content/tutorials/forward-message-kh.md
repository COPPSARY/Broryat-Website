ពេលអ្នកឃើញសារសង្ស័យក្នុង private chat ឬ group chat អ្នកអាច forward វាទៅ Broryat ដោយផ្ទាល់ ហើយទុកឱ្យ bot ជួយពិនិត្យជំនួសអ្នក។

## ៣ ជំហានងាយៗ

មិនចាំបាច់មាន dashboard ឬការកំណត់បន្ថែមឡើយ។ គ្រាន់តែបើក Telegram រួចផ្ញើមាតិកាសង្ស័យទៅកាន់ bot ជាការស្រេច។

### 01 — ស្វែងរក Bot ផ្លូវការ

បើកកម្មវិធី Telegram រួចប្រើប្រអប់ស្វែងរក (Search) នៅផ្នែកខាងលើ ដើម្បីវាយរកឈ្មោះ `@broryat_bot`។ សូមពិនិត្យ handle ឱ្យបានច្បាស់ ព្រោះជនបោកប្រាស់តែងបង្កើត bot ក្លែងក្លាយដែលមានអក្សរ ឬលេខបន្ថែម។ ចុចលើលទ្ធផលផ្លូវការ រួចចុច `Start` ដើម្បីចាប់ផ្តើមជជែក។ អ្នកគ្រាន់តែធ្វើម្តងគត់ បន្ទាប់មក bot នឹងនៅក្នុងបញ្ជីជជែករបស់អ្នកសម្រាប់ប្រើប្រាស់លឿន។

### 02 — ផ្ញើមាតិកាសង្ស័យ

Forward សារដោយផ្ទាល់ពី chat ដែលអ្នកបានទទួល ចម្លងតំណភ្ជាប់ (Link) ផ្ញើជា text ឬ attach ឯកសារផ្ទាល់។ សូមធ្វើដោយ **មិនចាំបាច់ចុចបើក link ឬ file ជាមុនឡើយ**។ ការ forward រក្សាបរិបទដើម — អ្នកផ្ញើ, link និងឯកសារ — ដើម្បីឱ្យ bot អាចពិនិត្យបានត្រឹមត្រូវនូវអ្វីដែលអ្នកទទួល។ បើសារមាន link ប៉ុណ្ណោះ អ្នកអាច copy URL ហើយផ្ញើជា text ក៏បានដែរ។

### 03 — ទទួលបានលទ្ធផលភ្លាមៗ

ក្នុងរយៈពេលប៉ុន្មានវិនាទី bot នឹងឆ្លើយតបនូវកម្រិតហានិភ័យយ៉ាងច្បាស់ — សុវត្ថិភាព, សង្ស័យ, ឬគ្រោះថ្នាក់ — ព្រមទាំងមូលហេតុខ្លីៗ និងជំហានអនុវត្តជាក់ស្តែង។ ប្រើវាដើម្បីសម្រេចចិត្តថាគួរមិនអើពើ, block អ្នកផ្ញើ, ឬផ្ទៀងផ្ទាត់តាមប្រភពដែលទុកចិត្តបាន។ ពេលមិនច្បាស់ចិត្ត សូមរង់ចាំចម្លើយពី bot ជានិច្ចមុននឹងចុចអ្វីមួយ។

## គន្លឹះ

កុំចុច link ឬ file មុនពេល bot ឆ្លើយតប។ បើមានគេអះអាងថាឯកសារជា CV ឬឯកសារការងារ តែមានកន្ទុយឯកសារប្លែក សូមប្រាប់គេឱ្យផ្ញើជា PDF ធម្មតាឡើងវិញ តាមរយៈប្រភពដែលអ្នកទុកចិត្តបាន។
